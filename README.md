**Openshift Deployment**

- Openshift >= 4.6.x
- Openshift CLI (`oc`) >= 4.6.x

**Local Deployment**

- JDK 8 (Java 1.8)
- maven
- Postgresql Database Server >= 14
- IBM Websphere Traditional Server 8.5.5
